# Agent: Intake

**Purpose:** Accept a new property address, gather basic info from the agent/client, and scaffold a new property JSON file.

**Inputs:**
- Property address
- Agent contact details
- Basic property specs (beds, baths, sqft, price)
- Initial photo assets

**Outputs:**
- `data/properties/[slug].json` populated with intake data
- Asset folder created at `app/public/images/[slug]/`

**Status:** Not yet implemented — placeholder for Phase 2.
